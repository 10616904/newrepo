Installing Git

Step1: to check Git is installed or not
	go to terminal git --version

step2: Download and install Git

step3: signup and create a account on GitHub
	https://github.com/

step4: Add your GitHub email and username to git
	git config --global user.email "email id"
	git config -- gloabl user.name "github username"

Step5: start adding files and folders to Git- tracking the changes

Step6: commands 
	- in terminal to the location of the folder/location of the project
	- git init
	- git status
	- git add filename
	- git status
	- git add
	- git commit -m "added file"
	- git remote add origin "location of remote repo"
	- git push -u origin master
	- git --help
